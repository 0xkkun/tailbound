# 네임드 몬스터 밸런스 가이드

## 개요

네임드 몬스터는 일반 필드 몬스터와 보스 사이의 **중간보스급 엔티티**입니다.
- **스폰 시간**: 3분, 6분, 9분에 각각 2마리씩 랜덤 스폰
- **총 스폰 수**: 최대 6마리 (3분 2마리 + 6분 2마리 + 9분 2마리)
- **특징**: 빨간색 풀네임 표시, 강력한 원거리 공격 패턴, 높은 체력

## 공격 패턴

네임드 몬스터는 **3가지 원거리 공격 패턴**을 사용합니다. (현재 통일, 추후 각 네임드별 고유 패턴 추가 예정)

### 패턴 1: 기본 공격 (Basic Attack)
- **설명**: 플레이어를 향해 불꽃 투사체 단발 발사
- **쿨타임**:
  - 3분: 3.0초
  - 6분: 2.7초
  - 9분: 2.4초
- **사거리**: ~450px
- **특징**: 가장 빈번하게 사용

### 패턴 2: 5연발 난사 (Burst Attack)
- **설명**: 플레이어를 향해 불꽃 5발 연속 발사 (0.15초 간격)
- **쿨타임**:
  - 3분: 8.0초
  - 6분: 7.0초
  - 9분: 6.0초
- **사거리**: ~400px
- **특징**: 약간의 랜덤 각도 (±5도) 추가로 회피 난이도 증가

### 패턴 3: 원형 방사형 (Radial Attack)
- **설명**: 12방향으로 불꽃 동시 발사 (360도 커버)
- **쿨타임**:
  - 3분: 12.0초
  - 6분: 10.5초
  - 9분: 9.0초
- **사거리**: ~500px
- **특징**: 가장 강력하고 회피하기 어려운 패턴

## 시간대별 난이도 곡선

### 3분 네임드 (Low Tier)
- **플레이어 상태**: 레벨 3-5, 무기 1-2개
- **네임드 스탯**: 기본 스탯 (HP 1,600, 데미지 35, 투사체 20)
- **전략**: 중간보스 경험 제공, 경험치 보상으로 빠른 성장 유도

### 6분 네임드 (Medium Tier)
- **플레이어 상태**: 레벨 8-12, 무기 3-4개, 파워업 증가
- **네임드 스탯**: 중간 강화 (HP 2,080, 데미지 42, 투사체 25)
- **전략**: 플레이어 성장에 맞춰 난이도 증가, 긴장감 유지

### 9분 네임드 (High Tier)
- **플레이어 상태**: 레벨 15-20, 무기 5-6개, 강력한 파워업
- **네임드 스탯**: 최대 강화 (HP 2,560, 데미지 50, 투사체 30)
- **전략**: 10분 보스 전투 직전 최종 테스트, 최대 경험치 보상 (480 XP)
- **주의**: 데미지는 보스(60)보다 약하게 설정됨

## 밸런스 디자인 철학

### 1. **위치**: 일반 몹과 보스 사이의 중간 단계
- 상급령보다 **3.5-5.7배 높은 체력** (상급령 450 HP vs 네임드 1,600-2,560 HP)
- 보스보다 **훨씬 약함** (보스 225,000 HP vs 네임드 최대 2,560 HP)
- 경험치 보상은 **상급령의 3-4.8배** (상급령 100 XP vs 네임드 300-480 XP)

### 2. **스폰 타이밍**: 게임 진행의 리듬 브레이커
- 3분/6분/9분 주기로 등장하여 **단조로움 방지**
- 각 시간대마다 플레이어가 충분히 성장한 상태에서 도전
- 10분 보스 전투 전 마지막 준비 기회

### 3. **보상**: 위험 대비 높은 리턴
- 높은 경험치 보상으로 **레벨업 촉진**
- 체력 포션 드랍으로 **생존력 확보**
- 보스 전투 전 **빠른 성장 기회 제공**

### 4. **난이도 조절**: 시간대별 점진적 강화
- 3분: 첫 만남, 학습 단계
- 6분: 중간 도전, 긴장감 유지
- 9분: 최종 테스트, 보스 전투 준비

## 플레이어 대응 전략

### 초반 (3분 네임드)
- **권장 레벨**: 3-5
- **전략**: 기본 공격 위주로 견제, 원형 패턴은 거리 벌리기
- **무기**: 1-2개 (부적, 부채 등)

### 중반 (6분 네임드)
- **권장 레벨**: 8-12
- **전략**: AoE 무기 활용, 5연발 패턴 회피 숙련
- **무기**: 3-4개 (부적, 부채, 작두칼 등)

### 후반 (9분 네임드)
- **권장 레벨**: 15-20
- **전략**: 고화력 무기로 빠른 처치, 넉백 활용
- **무기**: 5-6개 (모든 무기 조합)

## 추후 개선 계획

### 고유 공격 패턴 (각 네임드별)
1. **홍각추 도깨비**: 돌진 공격 + 광역 충격파
2. **한월령**: 냉기 슬로우 + 빙결 장판
3. **암혼령**: 어둠 장판 + 시야 제한
4. **희구령**: 순간이동 + 기습 공격
5. **혈미호왕**: 매혹 디버프 + 분신술
6. **고목장**: 나무 소환 + 속박
7. **심연객**: 물 장판 + 이동속도 감소

### AI 개선
- 플레이어 거리에 따른 패턴 선택 (근거리/중거리/원거리)
- 체력 단계별 패턴 변화 (50% 미만 시 공격 강화)
- 협동 공격 (2마리가 동시에 스폰되므로 조합 패턴)

### 시각 효과
- 스폰 시 등장 연출 (화면 흔들림, 경고 메시지)
- 처치 시 특별한 사망 이펙트
- 빨간색 이름 외 추가 시각 강조 (오라, 발광 효과)

## 주요 설정 값

### 밸런스 설정 (`balance.config.ts`)
```typescript
NAMED_ENEMY_BALANCE = {
  low: { health: 1600, damage: 35, projectileDamage: 20 },
  medium: { health: 2080, damage: 42, projectileDamage: 25 },
  high: { health: 2560, damage: 50, projectileDamage: 30 },
}

NAMED_SPAWN_TIMES = [180, 360, 540] // 3분, 6분, 9분

// 플레이어 DPS: 10분 시점 약 1000/초
// 보스: 체력 225,000 HP / 데미지 60 (약 3.75분 전투)
// 네임드: 체력 1,600-2,560 HP / 데미지 35-50 (약 1.6-2.6초 전투)
// 네임드 데미지는 보스보다 약하게 설정 (보스의 58-83%)
```

### 스폰 로직 (`NamedSpawnSystem.ts`)
- 각 시간대마다 **2마리씩 랜덤 타입** 선택
- 화면 밖 600px 거리에서 스폰
- 플레이어 타겟 자동 설정
